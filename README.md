# Traffic Flow Prediction Using Deep Learning

## Overview

Traffic Flow Prediction is a deep learning-based project that analyzes historical traffic data and predicts future traffic flow patterns. The project evaluates and compares the performance of RNN, GRU, and LSTM models to identify the most effective approach for traffic forecasting.

## Features

- Traffic flow prediction using historical data
- Comparative analysis of RNN, GRU, and LSTM models
- RMSE-based performance evaluation
- Traffic trend visualization
- Area-wise traffic forecasting
- Web-based interface for prediction and analysis

## Technologies Used

- Python
- Django
- RNN
- GRU
- LSTM
- HTML
- CSS

## Results

- Compared RNN, GRU, and LSTM models using RMSE.
- LSTM achieved the best prediction performance among the evaluated models.
- Generated traffic forecasts to support intelligent traffic management.
